## Setup

Clone the repo.

```sh
git clone git@github.com:MichaelCurrin/auto-tag.git
```


## Run

Make any changes.


Test the changes on the repo itself.

```sh
$ cd auto-tag/autotag
$ ./autotag --help
```

Or another repo.

```sh
$ cd other-repo
$ <PATH_TO>/autotag M --preview
```

Optionally add the script to your bin.

```sh
ln -s auto-tag/autotag ~/bin/autotag
```

```sh
autotag M --preview
```
