## Sample usage

Specify the tag increment level and the tool will figure out the next tag version and create the tag.

Here we increment the git version by bug/patch level, from `v0.1.0` to `v0.1.1`.

```sh
$ autotag b
```
```
Auto tagging...
Last tag: v0.1.0
New tag: v0.1.1
Creating annotated tag...
```
